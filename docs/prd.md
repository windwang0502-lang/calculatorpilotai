# 需求文档

## 1. 应用概述

### 1.1 应用名称
AI-Powered SEO Tool Website

### 1.2 应用描述
一个面向搜索引擎优化的工具网站平台，提供多类别实用工具（金融、健康、时间、AI、物流），每个工具均配备AI增强分析能力，通过内容优化和内部链接体系获取自然流量，可扩展为工具工厂模式。

### 1.3 核心目标
- 通过Google SEO获取自然流量
- 提供5个核心类别的实用计算工具
- 为每个工具结果提供AI驱动的洞察与建议
- 建立完整的内部链接生态（工具-工具、工具-指南、工具-对比）

---

## 2. 用户与使用场景

### 2.1 目标用户
- 需要快速计算工具的普通用户（房贷、BMI、年龄等）
- 寻求AI辅助决策建议的用户
- 通过搜索引擎查找特定工具的流量用户

### 2.2 核心使用场景
- 用户通过搜索引擎搜索「mortgage calculator」等关键词进入工具页面
- 用户输入参数后获得计算结果及AI分析建议
- 用户通过相关工具链接或指南内容深度浏览站内其他工具

## 3. 页面结构与功能说明

### 3.1 页面层级结构

```
首页 (/)
├── 工具分类页 (/tools/{category})
│   ├── 金融工具 (/tools/finance)
│   │   └── 房贷计算器 (/tools/finance/mortgage-calculator)
│   ├── 健康工具 (/tools/health)
│   │   └── BMI/卡路里计算器 (/tools/health/bmi-calorie-calculator)
│   ├── 时间工具 (/tools/time)
│   │   └── 年龄计算器 (/tools/time/age-calculator)
│   ├── AI工具 (/tools/ai)
│   │   └── AI检测器 (/tools/ai/ai-detector)
│   └── 物流工具 (/tools/shipping)
│       └── 材积重量计算器 (/tools/shipping/dim-weight-calculator)
├── 指南页面 (/guides)
│   └── 指南详情页 (/guides/{slug})
└── 对比页面 (/compare)
    └── 对比详情页 (/compare/{slug})
```

### 3.2 首页 (/)

**功能说明**
- 展示网站核心价值主张与定位
- 按类别展示5个核心工具入口（finance, health, time, ai, shipping）
- 提供搜索或导航功能快速定位工具
- 展示热门工具或最新指南内容

### 3.3 工具分类页 (/tools/{category})

**功能说明**
- 展示该类别下所有工具列表
- 每个工具显示名称、简短描述、使用场景
- 提供类别介绍与SEO优化内容

### 3.4 工具详情页（统一结构）

**URL规则**
- `/tools/{category}/{tool}`
- 示例：`/tools/finance/mortgage-calculator`

**页面组成（自上而下）**

#### 3.4.1 标题与SEO介绍区
- H1标题：工具名称（如「Mortgage Calculator」）
- SEO优化介绍文本：说明工具用途、适用场景、核心价值

#### 3.4.2 输入表单区
- 根据工具类型展示对应输入字段
- 输入字段包含标签、占位符、单位说明
- 提供「计算」按钮触发计算逻辑

#### 3.4.3 结果展示区
- 计算完成后显示核心结果数据
- 结果以清晰的数值、图表或分类形式呈现

#### 3.4.4 AI洞察区
- 基于计算结果输出AI分析内容
- 包含风险提示、优化建议、负担能力评估等

#### 3.4.5 FAQ区
- 展示至少5个常见问题与答案
- 问题与工具使用场景强相关

#### 3.4.6 相关工具区
- 展示3-5个相关工具链接
- 链接指向同类别或跨类别的其他工具

### 3.5 五个核心工具详细定义

#### 3.5.1 房贷计算器 (/tools/finance/mortgage-calculator)

**输入字段**
- 贷款金额
- 年利率
- 贷款年限

**输出结果**
- 月供金额
- 总利息
- 总还款额

**AI洞察内容**
- 负担能力建议（基于月供占收入比例的通用标准）
- 利率优化建议
- 还款策略提示

#### 3.5.2 BMI/卡路里计算器 (/tools/health/bmi-calorie-calculator)

**输入字段**
- 身高
- 体重
- 年龄
- 性别
- 活动水平

**输出结果**
- BMI数值
- 健康状态分类（偏瘦/正常/超重/肥胖）
- 每日卡路里需求估算

**AI洞察内容**
- 健康风险提示
- 饮食与运动建议
- 目标体重建议

#### 3.5.3 年龄计算器 (/tools/time/age-calculator)

**输入字段**
- 生日日期（用于计算当前年龄）
- 或两个日期（用于计算日期差）

**输出结果**
- 年龄（年、月、日）
- 或日期差（天数、周数、月数、年数）

**AI洞察内容**
- 生命阶段特征描述
- 时间管理建议

#### 3.5.4 AI检测器 (/tools/ai/ai-detector)

**输入字段**
- 文本内容（用户粘贴或输入）

**输出结果**
- AI生成概率（百分比）
- 分类结果（人类撰写/AI生成/混合）

**AI洞察内容**
- 检测依据说明（基于启发式规则）
- 文本特征分析
- 使用场景建议

#### 3.5.5 材积重量计算器 (/tools/shipping/dim-weight-calculator)

**输入字段**
- 长度（L）
- 宽度（W）
- 高度（H）
- 单位选择（厘米/英寸）

**输出结果**
- 材积重量（公式：L×W×H/139）

**AI洞察内容**
- 包装优化建议
- 运费节省策略
- 尺寸调整建议

### 3.6 指南页面 (/guides)

**功能说明**
- 展示所有指南文章列表
- 每篇指南显示标题、摘要、发布时间
- 指南内容与工具使用场景相关（如「如何选择合适的房贷方案」）

**指南详情页 (/guides/{slug})**
- 展示完整指南内容
- 内容中嵌入相关工具链接
- 提供相关指南推荐

### 3.7 对比页面 (/compare)

**功能说明**
- 展示对比文章列表
- 对比内容聚焦工具选择、方案对比（如「固定利率 vs 浮动利率」）

**对比详情页 (/compare/{slug})**
- 展示对比内容
- 内容中嵌入相关工具链接
- 提供相关对比推荐

## 4. 业务规则与逻辑

### 4.1 工具计算引擎规则

**纯函数架构**
- 每个工具的计算逻辑独立封装
- 输入参数 → 计算逻辑 → 输出结果
- 计算过程无副作用，可复用

**计算规则示例**
- 房贷月供计算：基于等额本息公式
- BMI计算：体重(kg) / 身高²(m)
- 材积重量：(L×W×H) / 139
- AI检测：基于启发式规则（词汇复杂度、句式模式等）

### 4.2 AI洞察生成规则

**触发时机**
- 用户完成计算并获得结果后自动生成

**生成内容类型**
- 风险提示：基于结果数值的风险等级判断
- 优化建议：针对用户输入参数的改进方向
- 决策支持：提供行动建议或参考标准

**内容生成方式**
- 基于预设规则与阈值生成结构化内容
- 示例：BMI > 30 → 输出「肥胖风险提示」+「减重建议」

### 4.3 SEO内容生成规则

**Meta信息**
- 每个工具页面生成唯一的title、description、keywords
- title格式：「{工具名称} - Free Online Tool」
- description包含工具用途、核心功能、使用场景

**结构化数据（JSON-LD Schema）**
- WebApplication类型：描述工具应用信息
- FAQ类型：标记FAQ区的问题与答案

**H1/H2层级结构**
- H1：工具名称
- H2：输入区、结果区、AI洞察、FAQ、相关工具

### 4.4 内部链接规则

**链接类型**
- 工具 → 工具：相关工具区展示同类别或跨类别工具
- 工具 → 指南：工具页面底部或侧边栏推荐相关指南
- 工具 → 对比：工具页面推荐相关对比文章
- 指南 → 工具：指南内容中嵌入工具链接
- 对比 → 工具：对比内容中嵌入工具链接

**链接选择逻辑**
- 基于类别相关性（同类别优先）
- 基于使用场景相关性（如房贷计算器 → 贷款指南）
- 每个页面至少包含3个内部链接

### 4.5 AI内容系统规则

**generateSEOContent(tool)功能**
- 输入：工具标识符
- 输出：SEO介绍文本、FAQ内容、Meta描述

**内容生成要求**
- SEO介绍：150-300字，包含关键词、使用场景、核心价值
- FAQ：至少5个问题，覆盖工具使用、结果解读、常见疑问
- Meta描述：120-160字符，简洁描述工具功能

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户输入非法字符（如负数、字母） | 输入框实时校验，提示正确格式 |
| 用户未填写必填字段 | 点击计算按钮后提示「请填写所有必填字段」 |
| 计算结果超出合理范围 | 显示结果但在AI洞察中提示「结果异常，请检查输入」 |
| AI检测器输入文本过短（<50字） | 提示「文本过短，建议至少输入50字以获得准确结果」 |
| 用户访问不存在的工具URL | 显示404页面，推荐热门工具 |
| FAQ内容未加载 | 显示占位文本「FAQ内容加载中...」 |
| 相关工具链接为空 | 隐藏相关工具区或显示「暂无相关工具」 |

## 6. 验收标准

1. 用户通过搜索引擎搜索「mortgage calculator」进入房贷计算器页面
2. 用户输入贷款金额、年利率、贷款年限后点击「计算」按钮
3. 页面显示月供金额、总利息、总还款额
4. AI洞察区自动显示负担能力建议与优化建议
5. 用户滚动至FAQ区查看常见问题
6. 用户点击相关工具区的「BMI Calculator」链接跳转至健康工具页面
7. 用户在BMI计算器页面输入身高、体重等信息完成计算
8. 用户点击页面底部的指南链接进入「如何选择合适的房贷方案」指南页面
